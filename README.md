# Guidebooks Hub

Центральный проект-**дирижёр** для оркестрации 5 туристических сайтов-путеводителей. Сами сайты живут в отдельных GitHub-репозиториях (форки одного движка Astro). Хаб не дублирует их логику — он ими управляет: запускает наполнение и публикацию, хранит мастер-документацию, стандарты и календари, проверяет достоверность кросс-сайтово.

## Структура
- `docs/SITES-REGISTRY.md` — реестр сайтов: репо, домены, гео, состояние.
- `docs/STANDARDS.md` — стандарты контента/фото/новостей (подтверждены).
- `docs/MASTER-PLAN.md` — план из 6 этапов, порядок, статус-трекер.
- `docs/audit/` — отчёты аудита состояния.
- `prompts/` — готовые промпты для вставки в Claude каждого репозитория.
- `sites/` — локальные клоны 5 репозиториев (в git не коммитятся).
- `scripts/clone-sites.sh` — склонировать/обновить все 5.
- `.claude/` — оркестраторы `/news`, `/content`, агенты хаба (добавляются на Этапе 5).

См. `docs/MASTER-PLAN.md` для текущего шага.

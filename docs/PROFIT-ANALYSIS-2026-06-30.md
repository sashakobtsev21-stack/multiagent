**Вывод для нас:** «Ezoic как первая ступенька» больше нерелевантна. Реальная лестница теперь:

Это хорошая новость: **порог входа в качественную дисплей-сеть упал в ~4 раза** (Raptive 100k→25k, Mediavine 50k sessions → $5k денег), и **наша EN-first US/UK/DE/NL/AU-аудитория прямо бьёт в условие «≥50% Tier-1»** — то есть мы квалифицируемся в Raptive/премиум быстрее, чем сайт со «смешанным» гео. RPM Tier-1 травел-контента в Journey по реальным отчётам: ~$3–5 на старте → **$8–12 RPM к 90 дням**; на 10–15k сессий это ~$50–175/мес, далее линейно. **Гео важнее объёма** — поэтому EN-first решение владельца напрямую ускоряет деньги.

## 1. Путь к ДИСПЛЕЙ-рекламе: какие EN-кластеры быстрее гонят Tier-1 объём к порогам

Логика: к 25k pv (Raptive) и $5k/год (Mediavine) тянут **верх-воронки informational-хабы** (большой объём, US/UK/DE/NL/AU-интент, длинная сессия), а НЕ узкие transactional-страницы (их ценность — в affiliate, а не в показах). Порядок публикации — «магниты объёма» вперёд.

**Кластеры-локомотивы объёма (Tier-1, быстрый климб к порогам):**

1. **Itineraries / how-many-days / road trip** (все три рынка) — самый большой Tier-1 объём, длинные сессии, раздаёт внутренние ссылки. `7/10-day itinerary`, `how many days in…`, `road trip`. Это №1 топливо для RPM-порога.
2. **Capital hub — things to do** (Sarajevo / Yerevan / Belgrade) — высокий объём, высокий dwell-time, перелинковочный центр.
3. **Best time to visit / weather by month** — стабильный сезонный Tier-1 объём, верх воронки, почти чистый показ-трафик (часто 💰 нет — но это и есть «дисплейные» страницы).
4. **Day-trips / iconic sights** — Mostar day trip (BIH), Khor-Virap-Noravank-Tatev (ARM), Iron Gate / day trips from Belgrade (SRB): объём + параллельно affiliate.
5. **Food & culture** (cevapi / khorovats-wine / rakija) — Pinterest/Tier-1 вовлечение, дешёвый ранг, длинные сессии, отлично перелинковывается.
6. **Comparisons** (`Bosnia vs Croatia`, `Armenia vs Georgia`, `Serbia vs Croatia`) — перехват на стадии выбора, Tier-1, верх воронки.

**Рынок-ранжирование по скорости климба к дисплей-порогам (быстрее всего Tier-1 объём):**
- **Serbia — самый быстрый.** Белград = бренд «party capital», дешёвые прямые рейсы Wizz/Air Serbia из UK (£28–50, лет 2.5 ч), огромный city-break + nightlife объём, EXIT. Найтлайф/«worth visiting»/«weekend break» — массовый UK-интент с коротким циклом решения → быстрый трафик к 25k pv.
- **Bosnia — средне-быстрый.** Гонит трафик за счёт **перехвата хорватского спроса** (Dubrovnik/Split→Mostar day trip — один из самых объёмных коммерч-узлов Балкан) + Sarajevo war-history (сильный эмоциональный Tier-1 магнит). Объём есть, но часть — day-tripper из Хорватии (короткая сессия).
- **Armenia — медленнее по «чистому» Западу, но глубже вовлечение.** Меньше прямого UK-рейс-спроса (нет лоукост-вала как в BEG/Балканы), но сильные USP-магниты (Mount Ararat, древнейшее вино, первая христианская страна) дают длинные сессии и лояльный hiking/culture-Tier-1. Climb к порогам чуть дольше; компенсируется ОЧЕНЬ сильным affiliate (см. §2) и ru-кошельком (§3).

**Перф-бюджет (обязателен для дисплей-RPM и порогов):** дисплей-реклама бьёт по Core Web Vitals → деньги напрямую зависят от скорости.
- Лимит на тяжесть рекламного слоя: **ленивые слоты, без CLS** (зарезервированная высота под объявления), sticky-футер-ад только если не рушит LCP.
- Сайты на Astro/Cloudflare (наш стек) — уже быстрые; беречь это: фото **webp ≤200 КБ** (уже в стандартах), без тяжёлых эмбедов, ad-скрипты — после контента.
- На входе (до Journey/Raptive) — **AdSense aутто-ads держать в узде**, чтобы не убить CWV до прихода нормальной сети.

**Consent Mode v2 (обязателен, EU-трафик DE/NL/скандинавы):** без корректного CMP + Google Consent Mode v2 дисплей-RPM по EU-трафику падает/обнуляется, и сети режут.
- Паттерн уже отработан в сети (Хорватия → порт на Грузию, см. задачи P1-1). **Тот же CMP/Consent v2 + CSP портировать на все 3 новых сайта на старте**, а не потом.
- В бэклоге сети уже висит privacy/cookie-policy страница + ссылка из баннера (задача #11) — для новых сайтов сделать сразу (требование Consent v2 и условие приёма в Mediavine/Raptive).

**Итог по доступности:** весь набор работает по трём рынкам. **Две дыры:** (1) **bikesbooking в Армении — только велосипеды** → байк-кластер для ARM не делаем как доходный, в BIH/SRB — делаем; (2) **kiwitaxi в Армении** скорее межгород, чем аэропорт → перед `zvartnots-airport-to-yerevan` transactional-страницей проверить, что трансфер реально бронируется, иначе вести её на trip-tours/aviasales.

**Какие связки конвертят — по рынкам:**

**Bosnia — деньги в day-trip-узле и авто:**
1. **trip-tours + kiwitaxi** на узле **Dubrovnik/Split → Mostar** (самый объёмный коммерч-узел страны; перехват хорватского трафика → тур или трансфер). №1 кошелёк BIH.
2. **localrent + safetywing** — «страну не раскрыть без авто» (петля Sarajevo→Mostar→Herzegovina); car-rental + франшиза/страховка; узел «which airport» (SJJ/Tuzla/Mostar) тянет aviasales+kiwitaxi.
3. **trip-tours** — Sarajevo war tour / Tunnel of Hope (высокий эмоциональный интент).
4. **trip-tours** — рафтинг (Una/Tara/Vrbas) — дорогой приключенческий продукт, слабо покрытый конкурентами.
5. **yesim / safetywing** — eSIM/страховка (низковоронные, высокомаржинальные).

**Armenia — деньги в day-tour и авто, ОЧЕНЬ сильный affiliate-профиль:**
1. **trip-tours** — массовый продукт **Khor Virap–Noravank–Tatev / Wings of Tatev** (~$45/чел, group-tour через Ереван) + Garni-Geghard. Армения почти не имеет общественного транспорта к монастырям → «нужна машина ИЛИ тур» = чистая affiliate-боль.
2. **localrent + safetywing** — авто = ядро самостоятельного маршрута (Zvartnots ~$32–42/день).
3. **yesim** — Армения один из самых дешёвых eSIM-рынков (от ~$0.60) → высокая конверсия.
4. **kiwitaxi / aviasales** — Yerevan↔Tbilisi (Кавказ-комбо) — перехват combo-трафика.
5. **trip-hotels / trip-tours** — winery (Areni/Vayots Dzor), brandy — доходный нишевый продукт с USP.

**Serbia — деньги в найтлайфе/stag, авто и Дунай-day-trips:**
1. **trip-hotels + trip-tours + kiwitaxi** на **Belgrade stag/bachelor + nightlife/splavovi** — высокобюджетная ниша (€500–1200/чел за уикенд), конвертит сразу в отели+трансфер+туры; уникальный сильный бренд. №1 кошелёк SRB.
2. **localrent + safetywing** — car-rental; раздел про **IDP** (US/non-EU нужен международный пермит) закрывает реальную боль и оправдывает авто-affiliate.
3. **trip-tours + localrent** — Iron Gate / Golubac / day trips from Belgrade (хаб-город → весь day-trip спрос).
4. **trip-hotels** — `where to stay in Belgrade` (прямой transactional) + 3-day Belgrade.
5. **yesim / safetywing** — eSIM/страховка.

**Сквозной приём (как у работающих сайтов):** партнёрские ссылки только через `/go/{partner}` с `rel="sponsored nofollow noopener"`; itinerary-верх воронки раздаёт внутренние ссылки на денежные страницы (авто/отели/туры); каждая transactional-страница = 1 намерение (анти-каннибализация, гейт ④).

## 3. ru-релокация как отдельный affiliate-кошелёк (Armenia + Serbia)

Армения и Сербия — **топ-направления ru-релокации с 2022** (общий язык де-факто, ВНЖ через ИП за 1–3 мес, низкие налоги, лояльные банки). Это **отдельный кошелёк**: ≈0 для дисплей-рекламы (ru-трафик = Tier-3 RPM, почти не платит за показы), но **очень высокий чек на affiliate** — релокация-сервисы, открытие банковского счёта, ДМС/insurance, долгосрочная аренда. По Боснии ru-кластера нет (сайт EN-only).

**Почему это отдельный трек, а не «ещё язык»:**
- Аудитория и аффилейт-набор ДРУГИЕ: не localrent/aviasales, а **релокация-агентство / банк / страховка / long-stay аренда** (высокий LTV лида).
- Деньги тут — НЕ от показов, а от лида: один оформленный ВНЖ/счёт/ДМС стоит на порядок дороже клика по рекламе. Поэтому ru-кластер оправдан, даже если рекламная RPM по нему нулевая.
- **YMYL + фактура year-specific** → жёсткий фактчек обязателен (ставки/пошлины/лояльные банки меняются ежегодно).

**Тайминг (важно — окна закрываются):**
- **Армения:** реформа иммиграционного закона **с 1 ноября 2026** — успеть «подать ВНЖ по старым/простым правилам ДО даты» = горячий, конвертящий угол; обязательное медстрахование планируется с 2027 (драйвер insurance). ru-кластер ~12–15 тем (ВНЖ/ИП/5% налог/банки/аренда Еревана/стоимость жизни).
- **Сербия:** ВНЖ через ИП/паушал, банки (Poštanska štedionica/API/OTP), ДМС для ВНЖ; ~12 тем. Фактура (паушал-пороги ~6 млн динар/год, госпошлины) устаревает → актуализируемый траст-хаб.

**Дисциплина запуска ru-кластера (фаст-фоллоу, НЕ на старте):**
- Сначала EN-ядро набирает траст (и дисплей-порог), потом ru-кластер отдельным языковым треком (паритет slug, hreflang + x-default — как в стандартах).
- Перед публикацией — перепроверка ставок/сроков у первоисточников (Минюст/налоговая/банки), кросс-фактчек (YMYL-гейт).
- Affiliate ru — точечный (релокация/банки/insurance/long-stay), без рекламной нагрузки на эти страницы (ru-показы не платят — не портить ими CWV/консент-картину EN-ядра).

## 4. Быстрый климб денежных кластеров (план очередности на ~первые 8–10 недель)

Принцип: **сначала informational-локомотивы объёма** (тянут к дисплей-порогам Raptive 25k / Mediavine $5k и раздают внутренние ссылки), **сразу с встроенными affiliate-блоками** на денежные страницы, чтобы affiliate начал капать ещё ДО дисплея. Один материал — полностью до публикации (гейты ①–④ + qa=GO), потом следующий (память finish-one-item-before-next).

**Волна 1 — магниты объёма + опорные деньги (нед. 1–3):**
- 3× **7-day itinerary** (BIH/ARM/SRB) — топливо порога + перелинковочный хаб (💰 localrent/trip-hotels).
- 3× **capital hub things-to-do** (Sarajevo/Yerevan/Belgrade) (💰 trip-tours/trip-hotels).
- 3× **best time to visit (month-by-month)** — сезонный объём (💰 trip-hotels).
- **SRB:** Belgrade nightlife + stag (💰 отели/трансфер/туры) — самый быстрый деньго-узел сети.
- **BIH:** Dubrovnik→Mostar day trip + things-to-do Mostar (💰 trip-tours/kiwitaxi) — самый объёмный коммерч-узел.
- **ARM:** Khor-Virap-Noravank-Tatev day tour + things-to-do Yerevan (💰 trip-tours).

**Волна 2 — transactional-деньги + day-trips (нед. 3–6):**
- 3× **car rental** (💰 localrent/safetywing) + 3× **driving in…**; узлы аэропортов (💰 aviasales/kiwitaxi).
- 3× **best eSIM for…** (💰 yesim — высокомаржинально, низкая воронка).
- **where to stay** (Belgrade/Sarajevo/Mostar/Yerevan) (💰 trip-hotels).
- Day-trip хабы: Iron Gate/day-trips-Belgrade (SRB), day-trips-from-Mostar (BIH), day-trips-from-Yerevan (ARM).

**Волна 3 — добор объёма и нишевой доход (нед. 6–10):**
- Food/wine/culture (вовлечение + перелинковка), nature/parks (рафтинг BIH, Sevan/Dilijan ARM, Tara/Stara Planina SRB), comparisons (перехват выбора), safety/budget/money (💰 safetywing/yesim).
- Сезонные ски/спа к осени (Jahorina/Bjelašnica, Tsaghkadzor, Kopaonik/Vrnjačka) — готовить за 2–3 мес до сезона.

**Параллелизм:** три сайта — три разных репо → можно вести параллельно (память one-writer-per-clone: один писатель на КЛОН, но между сайтами — параллель). ru-кластеры ARM/SRB — фаст-фоллоу после EN-ядра.

**Метрики-вехи монетизации:**
- **~1 000 сессий/сайт →** подключить **Journey by Mediavine** (70%) + AdSense-страховка; первый дисплей-доход.
- **25 000 pv/сайт при ≥50% Tier-1 →** заявка в **Raptive** (премиум-RPM) — наш EN-first профиль это условие выполняет.
- **$5 000 ad-rev/12 мес →** авто-апгрейд Journey→**Mediavine Official**.
- Affiliate — с первой опубликованной денежной страницы (не ждёт порогов); ru-кошелёк ARM/SRB — отдельным треком, высокий чек.

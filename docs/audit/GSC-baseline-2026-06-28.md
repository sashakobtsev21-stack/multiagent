## Ключевые сигналы
1. **Индексация работает, показы растут быстро.** Все 5 сайтов получают показы (4 «молодых» стартовали за последние 7 дней; Грузия +762%). Фундамент (индексация + появление в выдаче) — есть. Это главное, что подтверждает baseline.
2. **Новости — ранний двигатель трафика.** Топ-страницы по показам почти везде — НОВОСТИ: georgia `georgian-airways` 525 / `letniy-festival-batumi` 290 (6 кликов); montenegro `visa-russia` 177 (поз 4.55) / `flynas` 59 / `jazz` 34; albania `durres-folklore` (4 клика); macedonia `d-festival-dojran` 124 (6 кликов, поз 6.41). Новости индексируются быстро и ранжируются (свежесть + низкая конкуренция событийных запросов). → **еженедельный новостной дайджест оправдан, не убирать.**
3. **Огромный спрос на страховку, ещё не ранжируемся = деньги-возможность №1.** albania `travel insurance albania` (×4 варианта, 60+ показов) и croatia `croatia travel insurance` (×5, 100+ показов) — топ-запросы по показам, но **позиции 64–94** (страницы есть, ранг слабый). Страховка = 💰 аффилиат + высокий интент + доказанный спрос. → **усилить страховые статьи (глубина, перелинковка, свежесть) — топ-приоритет на климб с поз 70 → стр.1.**
4. **Гео подтверждает EN-first Tier-1.** 4 EN-сайта тянут США/UK/Германию/Австралию/Нидерланды — дорогой RPM, целевая аудитория. Грузия — CIS (Грузия/RU/UA) + неожиданно NL/US (EN-новости). Стратегия дорогого трафика работает. _(NL стабильно высоко везде — возможно дата-центр/бот-артефакт, не переоценивать.)_
5. **Клики низкие (84) — норма для недели №1.** Позиции в основном стр.3–10 (Google «песочница», ранг зреет 1–3 мес). Задача сейчас: публиковать + перелинковка + авторитет → позиции растут → клики придут. Не паниковать по кликам, следить за показами+позицией.

## Находки (severity · теги · доказательство · влияние · фикс · усилие · вердикт)

### [High 💰] EKTA — прямой URL без трекинга (потеря атрибуции дохода)
- **Доказательство:** `src/data/partners.json:33-36` — `"ekta": { "urlTemplate": "https://ektatraveling.com/", "allowSubId": false }`. В `$schema-note` задокументировано: присланная tpx.gr-ссылка вела на VisitorsCoverage с ошибкой «marker is not subscribed».
- **Влияние:** клики по страховке EKTA уходят без SubID → комиссия не атрибутируется, доход теряется. Страховка — по GSC топ-возможность №1 (см. ниже), так что это бьёт по самому денежному кластеру.
- **Фикс:** вступить в программу EKTA в Travelpayouts, взять корректную tpx.gr-ссылку, вернуть `?sub_id={subid}` + `allowSubId:true`. (Внешняя задача владельца — не код.)
- **Усилие:** S (правка одной строки JSON) + действие владельца в кабинете.
- **Вердикт:** известный задокументированный долг; держать как блокер монетизации страховки.

## 1I — Рекомендация каденс/KPI (по данным GSC, KPI = деньги/показы, не число статей)

- **KPI:** не «штук статей», а **показы + средняя позиция + доход по кластерам**. Следующий замер GSC — Пн 06.07.
- **Денежный приоритет №1 — СТРАХОВКА (climb с позиции 64–94 → стр.1).** `travel insurance albania` ×4 варианта, 60+ показов, ранг 64–94. Страницы есть (`albania-travel-insurance`, `do-you-need-insurance`, `/insurance/` хаб). Действия: углубить/освежить страховые статьи, усилить перелинковку с `is-albania-safe`/`is-albania-expensive`/`albania-visa`, **починить EKTA-атрибуцию**. Высокий интент + аффилиат + доказанный спрос = быстрый выигрыш.
- **Новости — держать недельный дайджест** (`durres-folklore` уже дал 4 клика; новости индексируются быстро на свежести/низкой конкуренции). НЕ слепой автопостинг (превью→«ок»→публикация).
- **Каденс:** план «3 статьи/будень на июль» (KALENDAR) — разумен как пропускная способность, но **темп подчинять KPI**: если страховой/entry-кластер растёт по показам — лить туда decision-stage и сателлиты, а не гнать вал новых тем. `albania-visa` (поз 8.31) и страховка — первоочередные цели усиления перелинковкой.
- **Недооценённый кластер:** entertainment/nightlife (`tirana-nightlife` есть) — по сети GSC показывает спрос; рассмотреть углубление при свободных слотах.

## 4. Деньги и трафик — что реально мешает цели и в каком порядке чинить

**Причинно-следственная цепочка (порядок неслучаен):**

```
ДОМЕН ЖИВОЙ + GSC  →  ИНДЕКСАЦИЯ  →  ОРГАНИЧЕСКИЙ ТРАФИК  →  ДОХОД (партнёрки + реклама)
   (блок A, P0)        (есть техн.       (контент/SEO          (EKTA-фикс, диверсификация,
                        обвязка)          уже готовы)            display-реклама — блок H)
```

- **Сейчас сеть «богата контентом, но, возможно, невидима».** Контент, SEO-кластеры, календари, перелинковка, гейты — всё это уже есть и качественно. Но если домены не привязаны и GSC не подключён, всё это не конвертируется в трафик. **Деньги не появятся, пока не закрыт блок A.**
- **Как только трафик пойдёт — деньги «утекают» из-за EKTA** (все 5) **и болванок Черногории**. Эти фиксы дешёвые (S) и должны идти сразу после/параллельно блоку A, чтобы первые же клики атрибутировались.
- **Дисплей-реклама** (явная цель владельца) — третий слой, имеет смысл только на живом трафике и требует решений (см. ниже).

### Блок H — Монетизация vs цель «заработок на рекламе» (стратегическая развилка)

Сеть монетизируется **только партнёрками** (`/go/` + AffiliateBox). **Дисплей-реклама (AdSense/Mediavine/Ezoic/Raptive) не подключена нигде.** В Грузии дисплей осознанно отклонён ранее (конфликт с CSP/перфом, near-zero RPM на ru/uk).

**Рекомендация по гео (вердикт «делать/не делать/когда»):**
- **Албания, Черногория, Хорватия, Македония (EN, Tier-1 EU/UK)** — здесь дисплей-реклама даёт реальный доход (высокий RPM в EUR/USD). **Вердикт: ДЕЛАТЬ, но ПОЗЖЕ** — после (1) живого трафика, (2) подключённого Consent Mode v2 (сейчас только у Хорватии). Пороги входа: AdSense — почти без порога (но низкий RPM); **Mediavine/Raptive — от 50k/100k сессий/мес**, Ezoic — без порога (компромисс по качеству). До достижения порогов — AdSense как старт ИЛИ ждать порога премиум-сети.
- **Грузия (ru/uk, СНГ)** — дисплей-реклама почти не окупается (near-zero RPM, риск для CSP/перфа). **Вердикт: НЕ ДЕЛАТЬ** для ru/uk; для en-трека Грузии — опционально вместе с остальными.

**Условия, без которых рекламу не включать:**
1. Consent Mode v2 + cookie-баннер на EU-сайтах (иначе юр-риск; сейчас только Хорватия).
2. Сохранить перф-бюджет (JS ≤50 КБ/стр, проверяется `qa.mjs`) и CWV — реклама не должна ронять Page Experience (просадка позиций → меньше трафика → меньше денег).
3. CSP придётся ослабить под рекламные домены — делать аккуратно, не открывая `script-src 'unsafe-inline'`.

**Affiliate-гигиена (делать сразу, дёшево):** починить EKTA (все 5) и болванки Черногории; для Грузии — начать диверсификацию от Trip.com. `/go/`-редирект безопасен (бьёт только по `partners.json` + fallback `/`) — подтверждено в шаблоне; `rel="sponsored nofollow noopener"` соблюдается.

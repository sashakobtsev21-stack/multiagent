# Аудит состояния 5 сайтов — 2026-06-22

Источник: параллельный скан 5 репозиториев (чтение реальных файлов через GitHub API). Ниже — факты на дату аудита.

## Сквозные находки
1. **Новости 10/2 дней работают во ВСЕХ 5** (`NEWS_SECTION_WINDOW=10`, `NEWS_HOME_WINDOW=2`, `daily-news-rebuild.yml` cron `0 5 * * *`). Чинить не нужно. Дыра — текст `/news` не локализован у Албании/Черногории/Македонии (грузинские источники).
2. **Календарь по дням есть только у Грузии** (`KALENDAR.md`). У 4 — только тематический `CONTENT_PLAN`.
3. **Города:** коллекция `cities` пустая ВЕЗДЕ, включая Грузию. Эталон: город = статья `category=goroda`.
4. **Недокопированный операционный слой:** у 4 нет `PROGRESS/HANDOFF/AUDIT` под свою страну (только грузинские копии в `docs/_georgia-reference/`); `docs/memory/MEMORY.md` пустой — а `CLAUDE.md` требует их вести.
5. **Домены:** живой только у Грузии; у Черногории/Хорватии/Македонии — плейсхолдеры (нет данных GSC).
6. **Монетизация:** Грузия — Trip.com ~84% (концентрация), EKTA сломана; Черногория — `/go/` физически не реализован.

## E. Стандарты и правило дисциплины (впиши в CLAUDE.md и/или CONTENT_GUIDE.md)

Впиши блок **СТАНДАРТЫ** (если уже частично есть — сверь и доведи до этой формулировки, не плоди дубли):

- **Фото (минимум):** статья **≥5** (cover + ≥4); новость **≥2** (cover + ≥1); маршрут — cover + фото **каждой** остановки; город **≥10**; столица **≥15**. Каждое фото уникально на материал, **webp ≤200 КБ**, только своё или CC/PD (Wikimedia/Openverse/Unsplash) с атрибуцией (автор + лицензия).
- **Город = статья `category: goroda`** (коллекция `cities` не используется).
- **Языки/паритет:** 3 языка en/ru/uk (для Грузии ведущие ru/uk, en — отдельный экспат/номад-трек); одинаковый slug, одинаковые числовые факты, взаимные hreflang + x-default.
- **Тон** живой, человечный, как у носителя языка (uk — живая мова, не суржик); **факты только из источника** (цены/адреса/часы + «уточняйте» + дата; нет данных → TODO + источник); **даты/числа — только из источника текущего года** (для ежегодных событий не копировать даты из прошлогодних статей).
- **Партнёрки только через `/go/{partner}`** с `rel="sponsored nofollow noopener"`.

## Дополнительные точечные находки (занеси в AUDIT и/или ROADMAP-FIX)

1. **EN-фолбэк внутренних ссылок.** Часть EN-статей в теле линкуется на ru-пути (фолбэк вместо английских URL). Учти: в этом движке **английский — корневая локаль (URL без языкового префикса)**, а ru→`/ru/`, uk→`/uk/`; при этом контент EN лежит в папке коллекции `src/content/articles/en/**`. **Сначала подтверди реальную схему английских URL** по `astro.config.*`/роутингу (не предполагай префикс `/en`), затем сделай **отдельный проход**: подними внутренние ссылки в EN-статьях на их корректные английские эквиваленты там, где английская страница существует. Где английского эквивалента ещё нет — оставь ru-ссылку и **занеси конкретный список (файл → недостающий EN-URL) в ROADMAP-FIX** (не выдумывай несуществующие URL). Это правка ссылок, а не перегенерация контента.
2. **EKTA-страховка сломана.** Для украинских туристов нужна EKTA, но в frontmatter страховых статей стоит `partner: 'safetywing'` (fallback). EKTA упоминается только в тексте (ru/uk), рабочего партнёрского пути под неё нет. Зафиксируй баг в **ROADMAP-FIX** (severity High): нужен партнёр EKTA в `src/data/partners.json` + корректный `/go/`-путь для украинской аудитории. **Сейчас не чини партнёрку** (это поздний этап) — только задокументируй.

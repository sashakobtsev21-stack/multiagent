Движок — общий форк: **Astro 6.4.6 (static/SSG)**, Tailwind 4, Leaflet (карты), деплой на **Cloudflare Workers** (деплой = `git push` в `main`). Общение и комментарии — по-русски. ## Контекст продукта
- Гео-приоритет: **EN-first, Tier-1 (Запад)**; **ru/uk вторичны**. Языки `en/ru/uk`; паритет: одинаковый slug, одинаковые числовые факты, взаимные `hreflang` + `x-default`.
- Валюта на сайте — **евро €** (у Черногории евро без собственного ЦБ).

- **Город = статья `category=goroda`**; коллекция `cities` НЕ используется (она пустая во всех форках, включая эталон).
- Зрелость репо — **частичная**: сейчас **5 статей × 3 языка** (`montenegro-travel-guide`, `best-time-to-visit`, `renting-a-car`, `cost-of-living`, `durmitor-national-park`), у каждой `cover + 3` инлайн-фото (= 4 фото). 0 городов / 0 маршрутов / 0 ресторанов / 0 услуг.
- **Механика новостей уже реализована в движке и работает** (раздел хранит 10 дней `NEWS_SECTION_WINDOW=10`, блок на главной — 2 дня `NEWS_HOME_WINDOW=2`, ежедневный rebuild `daily-news-rebuild.yml`). Её НЕ трогать и НЕ ломать — локализовать нужно только ТЕКСТ скилла `/news`.

- Источник правды по продукту — `SPEC.md`; правила — `CLAUDE.md` и `CONTENT_GUIDE.md`. Репозиторий на GitHub: `sashakobtsev21-stack/montenegro-site`.

Создай эту связку по образцу движка (рабочая в `croatia-site`/`gruzia-site`):

  4. `wrangler.jsonc` корректно ссылается на Worker и статику.
  Проверь сборку и поведение роута `/go/{partner}?c={slug}` локально (`wrangler dev` / `npm run preview` — что применимо в репо).

## СТАНДАРТЫ (вписать в `CLAUDE.md`/`CONTENT_GUIDE.md`)
- **Объём:** статья **1200–2000 СЛОВ** (первый абзац — прямой ответ за 2–3 предложения; H2/H3 под вопросы; без «воды» и без FAQ-блока); новость **1500–2000 ЗНАКОВ** (~250–350 слов, 3–5 разделов, **≥4 внутренних ссылки**).
- **Фото (минимум):** статья **≥5** (cover + ≥4); новость **≥2** (cover + ≥1); маршрут — cover + фото **КАЖДОЙ остановки**; город **≥10**; столица **≥15**. Каждое фото уникально на материал, формат **webp ≤200 КБ**, своё или **CC/PD (Wikimedia/Openverse/Unsplash) с атрибуцией** (автор + лицензия).
- **Город = статья `category=goroda`** (коллекция `cities` НЕ используется).
- **Тон:** живой, человечный, как у носителя языка (uk — живая мова, не калька/суржик); НЕ ИИ-генерёнка; факты **только из источника/брифа**; даты/время/числа — **только из источника текущего года** (для ежегодных событий не копировать прошлогодние даты).
- **Партнёрки — только через `/go/{partner}`** с `rel="sponsored nofollow noopener"`. Прямые партнёрские URL в контенте запрещены.

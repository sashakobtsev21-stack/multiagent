## E. Стандарты и правило дисциплины (вписать в `CLAUDE.md` и `CONTENT_GUIDE.md`)

Добавь явными разделами (не дублируй — если уже есть, приведи к этим формулировкам):

**Стандарты публикации:**
- Объём: статья — **1200–2000 слов** (первый абзац — прямой ответ за 2–3 предложения; H2/H3 под вопросы; без «воды» и без FAQ-блока); новость — **1500–2000 знаков** (~250–350 слов, 3–5 разделов, ≥4 внутренних ссылки).

Каждое фото уникально на материал, формат **webp ≤200 КБ**, своё либо CC/PD (Wikimedia/Openverse/Unsplash) с атрибуцией (автор + лицензия).
- Город = статья с `category=goroda` (коллекция `cities` для городов-путеводителей не используется по этой модели).
- Тон живой, человечный, как у носителя языка (uk — живая мова, не калька/суржик; en — как пишет носитель); НЕ ИИ-генерёнка. Факты — только из источника/брифа (цены/адреса/часы + «уточняйте» + дата; нет данных → TODO + источник). Даты/время/числа — только из источника текущего года.
